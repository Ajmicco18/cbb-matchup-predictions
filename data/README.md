# Data Folder 

This folder contains data downloaded from Kaggle["https://www.kaggle.com/competitions/march-machine-learning-mania-2026/data"] from their 2026 March Machine Learning Mania competition. It contains 35 .csv files with data from both men's and women's college basketball. However, due to the size of the data, it is ignored by the .gitignore file, but can be downloaded using the link above.
